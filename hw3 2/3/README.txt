ผลลัพท์ที่ได้และสิ่งที่ได้เรียนรู้จากการใช้โปรแกรมผลิด parser อัตโนมัติ คือ โปรแกรม parser อัตโนมัติจะทำการ parse ไฟล์ตัวอย่างของเรา ซึ่งในที่นี้คือไฟล์ simple.txt ตาม grammar ที่เรากำหนดในไฟล์ parser.y ในขณะที่ในขั้นตอนของการทำlexical analysis จะอยู่ในส่วนของโค่ด lexer ซึ่งทำหน้าที่จัดการกับระบบ token โดยใช้ regular expression ในการกำหนด อยู่ในไฟล์ parser.l หลังจากที่เราได้สร้างและรันไฟล์ทั้งสองที่ทำหน้าที่ในส่วนของ lexical analysis และ syntactic analysis แล้ว เมื่อเรานำไฟล์ตัวอย่างเข้าไปทดสอบ ผลที่ได้คือเมื่อ grammar ของไฟล์ตัวอย่างถูกต้องตามที่เราได้กำหนดไว้ จะมีการปริ้นเอาท์พุดประโยคออกมาเพื่อแสดงว่า code ในส่วนนั้นหรือในแต่ละบรรทัดถูกต้องตาม grammar ที่เราได้กำหนดไว้ ในขณะที่สมมติถ้าใน code มีบรรทัดที่ไม่ถูกต้องตาม grammar ที่เราได้กำหนดไว้ โปรแกรมก็จะมีการgenerateเอาท์พุดประโยคออกมาเพื่อแสดงว่า ในบรรทัดนี้เกิด syntactic error ขึ้น
